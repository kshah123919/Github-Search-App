# 🔍 GitHub User Search App

A simple and clean React application that allows users to search for GitHub profiles using the GitHub API.

---

## 🚀 Features

* 🔎 Search any GitHub username
* 👤 Displays user profile details:

  * Name
  * Username
  * Followers
  * Public repositories
* 🖼️ Shows profile avatar
* 🔗 Direct link to GitHub profile
* ⚡ Loading state handling
* ❌ Error handling (User not found)



---

## 🛠️ Tech Stack

* ⚛️ React (useState)
* 🎨 Tailwind CSS
* 🌐 GitHub REST API
* ⚡ Vite

---


## 💡 How It Works

* User enters a GitHub username
* On clicking **Search**, the app fetches data from GitHub API
* Displays user details dynamically
* Handles loading and error states
